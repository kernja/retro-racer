# Retro Racer

Written in App Game Kit Classic (v1.7), Retro Racer was inspired by classic racing games and was published for iOS, Android, and Windows platforms in 2013.

Retro Racer has:

* 9 different tracks
* Four gameplay modes
* Downloadable content
